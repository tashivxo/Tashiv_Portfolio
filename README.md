# Tashiv Govender — Portfolio

My portfolio website showcasing my work, skills, certifications, and experience as a Full Stack Software Developer. I made sure to make it interactive and also just wanted to have fun experimenting with different tools whilst also describing who I am and what I do

**Live site:** [https://tashivxo.github.io/Tashiv_Portfolio/](https://tashivxo.github.io/Tashiv_Portfolio/)

---

## Features

- **Animated UI** — Buttons feature magnetic hover, ripple click, and shine sweep effects powered by [anime.js](https://animejs.com/)
- **Staggered text reveals** — Page subtitles animate in character-by-character using anime.js
- **Smooth page transitions** — Section entrances powered by Framer Motion
- **Downloadable resume** — One-click PDF download from the About page
- **Responsive design** — Fully optimized for desktop, tablet, and mobile
- **Dark theme** — Sleek dark UI with cobalt blue accents

## Pages

| Page | Description |
|------|-------------|
| **Home** | Hero section with intro, social links, and call-to-action buttons |
| **About** | Bio, background info, and resume download |
| **Experience** | Professional work history and education timeline |
| **Skills** | Technologies and tools displayed in a card grid |
| **Projects** | Showcase of personal and professional projects |
| **Certifications** | Professional certifications with PDF links |
| **Contact** | Contact form and social/email links |

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS v4, Framer Motion, anime.js
- **UI Components:** shadcn/ui (Radix primitives)
- **Routing:** Wouter
- **Build Tool:** Vite
- **Deployment:** GitHub Pages
